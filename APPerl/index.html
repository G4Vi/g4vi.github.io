NAME

Actually Portable Perl (APPerl) - Build-once run-anywhere Perl using the cosmopolitan libc

DESCRIPTION

APPerl is a distribution of Perl for most x86_64 operating systems contained in a single binary. That's right, you can use the exact same binary to run perl on Windows, Linux, OS X, FreeBSD, OpenBSD, and NetBSD without a VM. It is even can be built with all default C extensions and contains the standard perl modules and scripts embedded inside of it as the binary format is also a ZIP file.

The extreme portability and ZIP container allows APPerl to be used to create cross-platform standalone perl applications. Right now, there isn't much tooling, but ideally it could be used as an alternative to building standalone executables with PAR Packer with the added benefits of greatly increased compatibility across x86_64 PCs. APPerl could also be embedded in development SDKs, carried on your USB drive, or just used a method to run the exact same perl on all your computers.

PERLDOC

APPerl includes the perldoc program. For example:

./perl.com /zip/bin/perldoc perlintro

In cmd.exe run perl.com without the leading ./
